# Lexical Semantics

词汇解释

1. Synonymy 同义词
2. Antonymy 反义词
3. Hypernymy 上义词
4. Meronymy 下义词

# WordNet

A lexical database of English.

# Word Similarity

- 路径长度：在WordNet中，两个词之间的路径长度可以用来衡量它们的相似性。
- 深度信息：在WordNet中，词的深度（在词汇树中的位置）也可以用来衡量词的相似性。
- 信息内容：词的信息内容（在语料库中的频率）也可以用来衡量词的相似性。
